# Demo rehearsal loop

## Core Move

Practice until the demo feels inevitable.

## Workflow

1. Fix the audience, room, time limit, equipment, and single belief the demo must change.
2. Write the narrative beats and identify the live artifact that proves each important claim.
3. Rehearse end to end while recording timing, hesitation, dead air, transitions, and failure points.
4. Remove weak beats, pre-stage fragile steps, and prepare a truthful fallback for every critical dependency.
5. Run a final cold rehearsal that finishes on time and lands the intended closing line.

## Output

Return a compact result with:

- Audience and belief shift
- Timed demo beats
- Failure and fallback map
- Revisions
- Cold-run verdict

## Guardrails

- Do not substitute slide polish for a working demonstration.
- Never conceal a mocked, cached, or pre-recorded step.
- A rehearsal is incomplete until it has been timed under realistic conditions.
