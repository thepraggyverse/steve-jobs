# Revenge motive check

## Core Move

Detect when strategy is secretly ego or revenge.

## Workflow

1. Write the proposed action, stated objective, target, and the personal injury or rivalry surrounding it.
2. Ask whether the action still creates user or organizational value if the rival never notices.
3. Compare the plan with the best neutral alternative using economics, mission fit, opportunity cost, and collateral harm.
4. Remove punitive, symbolic, or escalatory elements that do not improve the legitimate objective.
5. Delay irreversible action until an uninvolved reviewer can challenge the remaining rationale.

## Output

Return a compact result with:

- Action and emotional context
- Rival-free value test
- Neutral alternative
- Removed punitive elements
- Independent-review decision

## Guardrails

- Anger can reveal a boundary but should not set strategy alone.
- Do not weaponize employees, customers, partners, or public channels in a personal conflict.
- Use legal or safety escalation when genuine harm requires it.
