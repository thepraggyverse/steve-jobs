# Make something wonderful

## Core Move

Raise ambition, care, usefulness, and emotional quality.

## Workflow

1. Establish the artifact's current useful baseline and intended beneficiary.
2. Evaluate usefulness, care, clarity, emotional quality, and lasting value separately.
3. Find the moment where adequate execution most visibly falls short of the ambition.
4. Propose one high-leverage improvement that deepens both utility and felt care.
5. Define the next artifact and user reaction that would prove the improvement matters.

## Output

Return a compact result with:

- Useful baseline
- Ambition gap
- Wonder opportunity
- High-leverage improvement
- Proof of meaningful improvement

## Guardrails

- Wonder must grow from usefulness and care.
- Do not use ambition language to excuse an unfinished core experience.
- Do not use long quotations from the source material.
- Prefer one consequential improvement over a list of flourishes.
