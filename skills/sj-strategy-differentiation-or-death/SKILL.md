# Differentiation or death

## Core Move

Check whether the product is meaningfully different.

## Workflow

1. Define the target user, urgent alternative, and purchasing or adoption criteria.
2. List competitors and substitutes using the user's language rather than category labels.
3. Identify a difference that is valuable, perceivable, defensible, and supported by an operating capability.
4. Test whether the difference changes a real choice through comparison, usage, or willingness-to-pay evidence.
5. Concentrate product and message around the proven difference or stop funding the undifferentiated bet.

- User and alternatives
- Choice criteria
- Defensible difference
- Choice-changing evidence
- Concentrate-or-stop call

## Guardrails

- Novelty without user value is not differentiation.
- A claim competitors can copy in a sentence is not yet defensible.
- Do not invent category language to hide commodity economics.
