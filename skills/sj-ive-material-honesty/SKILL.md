# Material Honesty

## Core Move

Check whether the chosen materials, finishes, surfaces, and sensory cues express the real product idea instead of disguising it.

## Workflow

1. List the artifact qualities users can see, touch, hear, carry, or notice over time.
2. Name what each quality communicates about the product.
3. Find any mismatch between claimed value and material signal.
4. Identify one simpler or more honest material choice.
5. Specify what prototype or inspection would prove the change.

## Output

Return a compact result with:

- Sensory inventory
- Signals sent
- Mismatches
- Honest alternatives
- Prototype or inspection

## Guardrails

- Do not chase premium signals for their own sake.
- Respect constraints, durability, repair, and manufacturing reality.
- For software, translate material into motion, latency, type, spacing, sound, and interaction texture.
- Do not use long quotations from the source material.
