# First Touch Moment

## Core Move

Treat the first encounter as a trust-setting moment where care, hierarchy, affordance, and meaning must be felt immediately.

## Workflow

1. Describe the exact first moment in sequence.
2. Identify what the user learns before any explanation.
3. Find friction, uncertainty, roughness, or careless signals in that moment.
4. Design one more generous version of the first encounter.
5. Define the next physical, visual, or interactive artifact to test.
