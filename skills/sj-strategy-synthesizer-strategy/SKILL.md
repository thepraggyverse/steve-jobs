# Synthesizer strategy

## Core Move

Combine separate technologies into one coherent product.

## Workflow

1. Inventory the capabilities, technologies, insights, and assets available across domains.
2. Choose one user problem whose complete solution requires a non-obvious combination of them.
3. Define the integration mechanism and the unique whole that no component delivers alone.
4. Remove components that add novelty without improving the end-to-end outcome.
5. Prototype the combined experience and test whether users perceive one coherent product rather than a bundle.

## Guardrails

- Synthesis requires functional integration, not a list of partnerships.
- Respect licensing, attribution, and domain expertise.
- Complexity belongs behind the experience, not in the user's task.
